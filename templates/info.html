The Idea

This project began by scraping the Rarefilmm.com website for movie information using the Beautiful Soup library for python. Initially, we were going for some idea about the genre distributions of the films on the site in comparison to more objective sources, such as imdb, but eventually landed on the idea of a movie recommendation system.

We would take user input- a movie the user enjoys, and recommend movies from our more nich√© database based on it. This web application was a last-minute idea, as it seemed interesting to practice some HTML, CSS, and Python's Flask library.

Data Collection

As it turned out, to make a reasonable movie recommendation system, we needed much more than just the titles, release years, and genres that our scraped data contained. We found the TMDB API and got to work, using the API to collect the missing data for the movies, mostly by using the script 'collect_metadata.py' found in the project's files, filling in information such as the rating, poster-urls, overviews of the films.

The data took a lot of work to collect properly due to lots of inconsistencies in TMDB's output, getting the Rarefilmm data to conform with the TMDB entries and so forth. Most of the collection and cleaning was done in Jupyter Notebooks later into the project.
